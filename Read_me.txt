Steps to run the website Locally:

1) Download my git repo- (https://github.com/Aaryan-2k/xenon_stack)
2) Open repo folder in visual studio code
3) Type command in visual stdio code terminal- (pip install -r requirements.txt) 
4) Make sure you are using Python 3.12.1
5) Type command in visual stdio code terminal- (python app.py)
6) the website will be hosted locally at-

  http://127.0.0.1:5000
